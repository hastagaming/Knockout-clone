# 🥊 Knockout 2D Deluxe (React Edition)

Selamat datang di **Knockout 2D**, sebuah game arkade sederhana yang terinspirasi dari mekanik populer di Roblox. Game ini dibangun menggunakan **React** dan dijalankan langsung melalui browser tanpa perlu instalasi!

## ✨ Fitur Game
* **Mode VS AI:** Uji kecepatan jarimu melawan komputer yang tangguh.
* **Mode VS Player:** Main berdua dengan teman dalam satu keyboard.
* **Sistem Skor:** Pantau siapa yang paling sering melakukan Knockout.
* **Kontrol Responsif:** Support klik mouse dan tombol keyboard.

## ⌨️ Kontrol Game
| Pemain | Tombol Keyboard | Aksi |
| :--- | :--- | :--- |
| **Pemain 1 (Biru)** | `A` | Dorong ke Kanan |
| **Pemain 2 (Merah)** | `L` | Dorong ke Kiri |

## 🛠️ Teknologi yang Digunakan
* [React.js](https://reactjs.org/) - Library UI utama.
* [Babel](https://babeljs.io/) - Untuk kompilasi JSX di browser.
* [GitHub Pages](https://pages.github.com/) - Untuk hosting gratis.
